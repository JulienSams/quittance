# Plan 03-01: Install and Configure Date Picker Components

## Objective

Install shadcn calendar and popover components, configure French locale for react-day-picker, and verify components render correctly.

## Context

Phase 3 requires date selection UI using shadcn's Calendar component (which wraps react-day-picker). The calendar must support French locale for proper date formatting and display. Two separate date pickers will be used for start and end dates.

## Tasks

<task id="1">
Install shadcn calendar component using CLI:
```bash
npx shadcn@latest add calendar
```
Verify the component is created at `src/components/ui/calendar.tsx`
</task>

<task id="2">
Install shadcn popover component (required by calendar for dropdown):
```bash
npx shadcn@latest add popover
```
Verify the component is created at `src/components/ui/popover.tsx`
</task>

<task id="3">
Install react-day-picker with French locale support:
```bash
npm install react-day-picker date-fns
```
date-fns is needed for French locale configuration with react-day-picker.
</task>

<task id="4">
Create a test component to verify calendar works with French locale:
- Import Calendar, Popover components
- Configure fr locale from date-fns
- Render a simple date picker with "Sélectionner une date" label
- Test that French month names and day names appear correctly
- Can delete this test component after verification
</task>
