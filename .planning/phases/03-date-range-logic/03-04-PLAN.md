# Plan 03-04: Implement Prorata Calculation Logic

## Objective

Create prorata calculation function that computes rent and charges for partial months based on real days, following user's specified rules: real days in month, both first and last month, round to nearest cent, apply to both rent and charges.

## Context

When a month is partial (doesn't cover the full month), calculate proportional amounts using: `(amount / daysInMonth) × daysOccupied`. Apply to both loyerHorsCharges and charges. Round to nearest cent (standard rounding: ≥0.5 up, <0.5 down).

## Tasks

<task id="1">
Create src/lib/prorata-calculator.ts with prorata utilities:
- Function: `calculateProrataAmount(fullAmount: number, daysOccupied: number, daysInMonth: number): number`
- Formula: (fullAmount / daysInMonth) × daysOccupied
- Round result to 2 decimals using Math.round((amount * 100)) / 100
- Return the prorated amount in euros (cents precision)
</task>

<task id="2">
Add function to calculate days in a month:
- `getDaysInMonth(year: number, month: number): number`
- Use: new Date(year, month + 1, 0).getDate()
- Correctly handles leap years (February = 28 or 29)
- Correctly handles 30-day and 31-day months
</task>

<task id="3">
Add function to count days occupied in a period:
- `getDaysOccupied(startDate: Date, endDate: Date): number`
- Inclusive count: if start=15th, end=20th, that's 6 days (15,16,17,18,19,20)
- Use: Math.floor((endDate.getTime() - startDate.getTime()) / (1000 * 60 * 60 * 24)) + 1
</task>
