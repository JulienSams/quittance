# Plan 05-01: Batch PDF Export & ZIP Download

## Objective

Implement batch PDF generation and ZIP download functionality. Users click a button to generate all receipts in their selected date range, see progress during generation, and automatically download a ZIP file containing all PDFs with proper naming.

## Context

**User decisions from CONTEXT.md:**
- No preview functionality - go straight to batch generation
- Progress indicator showing "Generating... X/Y receipts"
- Fail entire batch on any error (no partial ZIPs)
- ZIP filename: `quittances_[tenant]_YYYY-MM_YYYY-MM.zip`
- PDF filenames: `quittance_YYYY-MM.pdf`
- Flat ZIP structure (no folders)
- Button placement: after date pickers section
- Auto-download after generation
- Full validation before generation starts

**Existing assets:**
- `generateReceiptPDF()` - generates single receipt PDF
- `monthlyReceipts` array - already calculated in ReceiptData
- `ReceiptForm` component - where button will be added
- `validation.ts` - can extend for batch validation

<task id="05-01-03">
<title>Create ZIP filename generator</title>
<wave>1</wave>

<prompt>
Add `generateZipFilename()` helper function to `src/lib/batch-generator.ts`.

**Function signature:**
```typescript
export function generateZipFilename(receiptData: ReceiptData): string
```

**Implementation:**
1. Extract tenant last name: `receiptData.locataire.nom`
2. Normalize for filesystem:
   - Convert to lowercase
   - Remove accents: `.normalize('NFD').replace(/[\u0300-\u036f]/g, '')`
   - Replace non-alphanumeric with underscore: `.replace(/[^a-z0-9]/g, '_')`
   - Collapse multiple underscores: `.replace(/_+/g, '_')`
   - Truncate to 30 chars: `.substring(0, 30)`
3. Format start date: `dateDebut` as YYYY-MM
4. Format end date: `dateFin` as YYYY-MM
5. Return: `quittances_${tenantName}_${startDate}_${endDate}.zip`

**Example output:**
- Input: "Dupont", 2026-01, 2026-12
- Output: "quittances_dupont_2026-01_2026-12.zip"

**Edge cases:**
- Names with accents: "François" → "francois"
- Names with spaces: "Le Blanc" → "le_blanc"
- Long names: truncate at 30 chars
</prompt>

<task id="05-01-05">
<title>Add batch generation button to ReceiptForm</title>
<wave>2</wave>

<prompt>
Add batch generation UI to `src/components/ReceiptForm.tsx`.

**1. Import dependencies:**
- generateReceiptBatch, generateZipFilename from '@/lib/batch-generator'
- validateBatchGeneration from '@/lib/validation'

**2. Add state:**
```typescript
const [isGenerating, setIsGenerating] = useState(false)
const [progress, setProgress] = useState({ current: 0, total: 0 })
const [generationError, setGenerationError] = useState<string | null>(null)
```

**3. Create handler:**
```typescript
async function handleGenerateBatch() {
  // Reset state
  setGenerationError(null)
  setProgress({ current: 0, total: 0 })

  // Validate
  const validation = validateBatchGeneration(receiptData)
  if (!validation.valid) {
    setGenerationError(validation.errors.join('. '))
    return
  }

  // Generate
  setIsGenerating(true)
  const total = receiptData.monthlyReceipts?.length || 0
  setProgress({ current: 0, total })

  try {
    const zipBlob = await generateReceiptBatch(
      receiptData,
      (current, total) => setProgress({ current, total })
    )

    // Download
    const filename = generateZipFilename(receiptData)
    const url = URL.createObjectURL(zipBlob)
    const link = document.createElement('a')
    link.href = url
    link.download = filename
    link.click()
    URL.revokeObjectURL(url)

    setIsGenerating(false)
  } catch (err) {
    setGenerationError(
      err instanceof Error ? err.message : 'Erreur lors de la génération des quittances'
    )
    setIsGenerating(false)
  }
}
```

**4. Add UI (after date range accordion section, before save button):**
```tsx
{/* Batch Generation Section */}
<div className="mt-6 space-y-2">
  <Button
    type="button"
    onClick={handleGenerateBatch}
    disabled={isGenerating || !receiptData.dateDebut || !receiptData.dateFin}
    className="w-full"
  >
    {isGenerating
      ? `Génération... ${progress.current}/${progress.total}`
      : 'Générer les quittances'
    }
  </Button>

  {generationError && (
    <p className="text-sm text-red-600">
      {generationError}
    </p>
  )}
</div>
```

**Placement:** After the last accordion section (date range), before any existing sticky save button.
</prompt>
