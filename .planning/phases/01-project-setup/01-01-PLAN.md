# Plan 01-01: Initialize Vite + React + TypeScript Project

## Objective

Bootstrap a React + TypeScript project using Vite with proper configuration for development and production builds.

## Context

From CONTEXT.md:
- Use Vite to initialize the project (fast dev, simple config)
- No routing needed (single-page app)
- Set up ESLint + Prettier from the start
- Professional setup for a frontend-only application

## Tasks

<task id="1">
Initialize Vite project with React + TypeScript template using `npm create vite@latest . -- --template react-ts`
</task>

<task id="2">
Install dependencies with `npm install`
</task>

<task id="3">
Configure Vite for optimal development experience:
- Update vite.config.ts with proper TypeScript paths
- Configure dev server port if needed
- Set up build optimizations
</task>

<task id="4">
Set up ESLint configuration:
- Install ESLint and necessary plugins (@typescript-eslint/parser, @typescript-eslint/eslint-plugin, eslint-plugin-react-hooks)
- Create .eslintrc.json with React + TypeScript rules
- Add lint script to package.json
</task>

<task id="5">
Set up Prettier configuration:
- Install Prettier and eslint-config-prettier (to avoid conflicts)
- Create .prettierrc with consistent formatting rules (2 spaces, single quotes, trailing commas)
- Add format script to package.json
</task>

<task id="6">
Update tsconfig.json for strict TypeScript:
- Enable strict mode
- Configure path aliases (@/ for src/)
- Set proper module resolution
</task>

<task id="7">
Clean up default Vite template:
- Remove default Vite + React assets and example code
- Create minimal App.tsx shell
- Update main.tsx if needed
</task>

<task id="8">
Verify project builds and runs:
- Run `npm run dev` to test dev server
- Run `npm run build` to test production build
- Confirm no TypeScript or ESLint errors
</task>

## Verification Criteria

- [ ] Dev server starts without errors (`npm run dev`)
- [ ] Production build completes successfully (`npm run build`)
- [ ] TypeScript compiles with no errors
- [ ] ESLint runs clean (`npm run lint`)
- [ ] Prettier formats code consistently (`npm run format`)
- [ ] Hot module replacement works in dev mode

## Must Haves

From Phase 1 goal: "Can create and build the project"

- Vite project initialized with React + TypeScript
- All dependencies installed correctly
- TypeScript configuration is strict and correct
- Code quality tools (ESLint, Prettier) are functional
- Project builds for both dev and production
