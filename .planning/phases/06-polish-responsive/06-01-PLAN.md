# Plan 06-01: Mobile & Visual Polish

**Objective:** Apply mobile-first responsive refinements and systematic visual polish to existing UI. Make heading responsive, apply semantic color tokens, establish consistent spacing rhythm, and add subtle section backgrounds.

---

## Context

Phase 6 polishes the functional app with professional UI refinements. This plan focuses on visual consistency and mobile optimization for screens < 640px.

**Key Insight:** App already mobile-friendly (accordion layout), so refinements are polish not restructuring.

**User Decisions:**
- Phone-first optimization (< 640px)
- Smaller heading on mobile (text-3xl → text-2xl/xl)
- Systematic spacing scale (space-y-4/6/8)
- Subtle backgrounds in accordion sections (bg-muted/5)
- Semantic error colors (text-destructive, border-destructive)
