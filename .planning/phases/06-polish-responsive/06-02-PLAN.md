<task id="06-02-02">
<summary>Add reset button with confirmation</summary>
<details>
Add reset/clear button next to save button with confirmation dialog before clearing form.

**Changes in src/components/ReceiptForm.tsx:**

1. Add reset button next to save button (in existing button container):
```tsx
<Button
  variant="secondary"
  onClick={handleReset}
  disabled={isSaving}
>
  Réinitialiser
</Button>
```

2. Add confirmation handler above return statement:
```tsx
const handleReset = () => {
  if (confirm('Êtes-vous sûr de vouloir réinitialiser le formulaire ? Cette action ne peut pas être annulée.')) {
    resetForm()
  }
}
```

**Why browser confirm():**
- Simple, native, no new dependencies
- Clear warning prevents accidental data loss
- Acceptable UX for MVP
- Can upgrade to custom modal later if needed

**Button Styling:**
- `variant="secondary"` — less prominent than primary save button
- Disable while saving to prevent conflicts
- Place after save button (save is primary action)

**Testing:**
- Click reset button
- Verify confirmation dialog appears
- Click Cancel — form should not reset
- Click OK — form should clear to defaults
- Verify works with saved data in localStorage
</details>
