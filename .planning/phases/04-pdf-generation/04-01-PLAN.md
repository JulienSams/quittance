# Plan 04-01: PDF Receipt Template & Generation

## Objective

Create a legally compliant PDF template component for French rental receipts and implement single receipt generation functionality. The template follows traditional French quittance format with all mandatory legal mentions per ALUR law.

## Context

- **Prior work**: @react-pdf/renderer integrated (Phase 1), MonthlyReceipt data structure ready (Phase 3)
- **User decisions**: Traditional French layout, centered title, two-column landlord/tenant, dark blue accent (#1e3a8a), generous whitespace
- **Integration**: Takes MonthlyReceipt + ReceiptData as input, outputs PDF blob for Phase 5 batch export
